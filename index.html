Router

Fungsi router adalah menghubungkan jaringan yang berbeda (seperti LAN di rumah/kantor ke internet/WAN), mengelola lalu lintas data dengan meneruskan paket data ke alamat IP yang benar (routing), mendistribusikan koneksi internet ke banyak perangkat (via Wi-Fi atau kabel), serta meningkatkan keamanan melalui fitur seperti firewall dan enkripsi.

Jenis-Jenis Router

  • Router Wired (Kabel)
  • Router Wireless (Nirkabel)
  • Router Core
  • Router Edge
  • Router Virtual
